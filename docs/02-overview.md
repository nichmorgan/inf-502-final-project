# Project Overview

## What
A **web application** for comparing GitHub repositories side-by-side

## Why
Understanding repository health, community engagement, and development patterns

## How
- **GitHub API** (PyGithub) for data collection
- **Clean Architecture** for maintainability
- **NiceGUI** for interactive web interface
- **Pickle + TTL** for smart caching

## Key Metrics
✅ Pull Requests (Open/Closed)  
✅ Contributors Count  
✅ Temporal Trends (1 year)  
✅ Multi-Repository Comparison
